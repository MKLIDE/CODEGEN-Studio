# Changelog

All notable changes to CodeGen Studio will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-01

### Added
- Initial release of CodeGen Studio Vision 1.0
- Local-first architecture with Tauri + Rust
- Embedded Java Spring Boot backend
- React TypeScript frontend with Monaco editor
- AI code suggestions using local llama.cpp
- Project templates for React, Node.js, Spring Boot
- Privacy features: network blocking, encrypted storage
- One-click project operations: run, test, export
- Cross-platform support: Windows, macOS, Linux

### Features
- Create projects from templates
- AI-powered code suggestions
- Real-time error checking
- Integrated terminal
- File explorer with tree view
- Settings panel with theme options
- Privacy monitoring dashboard
- Export projects as ZIP files

### Technical
- 85MB installer size target
- Startup under 3 seconds
- AI response under 4 seconds
- Zero critical security vulnerabilities
- 100% offline functionality
