Local-first architecture

Three layers, one mission: keeping your code private.

Frontend Layer

React + TypeScript + Vite

Modern, responsive UI that communicates only with local processes. Zero external API calls.

Desktop Shell

Tauri (Rust)

Lightweight, secure bridge between UI and system. No telemetry, no background phoning home.

Business Logic

Java Spring Boot (Embedded)

Local server process that runs entirely on your machine. All models and data stay local.

AI Engine

llama.cpp + Local Models

Inference runs on your hardware. Download models once, use them forever—offline.
