# User Guide

Welcome to CodeGen Studio! This guide will help you get started with our privacy-first development environment.

## Getting Started

### Installation

1. **Download** the latest release for your platform
2. **Install** using the provided installer
3. **Launch** CodeGen Studio

### First Launch

On first launch, you'll see:
- Welcome screen with setup options
- Privacy settings configuration
- Tutorial walkthrough

## Creating Projects

### Step 1: Choose Template
1. Click "New Project" button
2. Select a template:
   - **React + TypeScript**: Modern web apps
   - **Node.js + Express**: REST APIs
   - **Spring Boot**: Java applications
   - **React Native**: Mobile apps
   - **Vanilla JavaScript**: Simple projects

### Step 2: Configure Project
- Enter project name
- Choose location
- Select features
- Configure database (if applicable)

### Step 3: Generate Project
Click "Create" to generate your project with:
- Complete file structure
- Configuration files
- Development setup
- Example code

## AI Assistance

### Enabling AI
1. Go to Settings → AI
2. Enable "Local AI"
3. Download a model (optional)
4. Configure AI settings

### Getting Suggestions
1. Type code normally
2. Press Ctrl+Space for suggestions
3. Accept suggestions with Tab
4. Review alternatives

### AI Features
- **Code completion**: Context-aware suggestions
- **Code generation**: Generate functions/classes
- **Error detection**: Find potential issues
- **Refactoring**: Improve code structure

## Troubleshooting

### Common Issues

#### AI Not Working
1. Check if AI is enabled in settings
2. Ensure model is downloaded
3. Check system requirements

#### Project Won't Run
1. Verify dependencies are installed
2. Check project configuration
3. View terminal output for errors

#### Performance Issues
1. Close unused projects
2. Disable AI if not needed
3. Check system resources
